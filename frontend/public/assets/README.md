# Image Assets Directory

Place your custom website images here:

1. **Website Logo:**
   - Filename: `logo.png`
   - Path: `frontend/public/assets/logo.png`

2. **Hero Profile Photo:**
   - Filename: `hero.jpg` (or `hero.png`)
   - Path: `frontend/public/assets/hero.jpg` (or `frontend/public/assets/hero.png`)
